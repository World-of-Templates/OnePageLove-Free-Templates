== Description ==

Integral One Page Business Theme - Build a professional website for your business with Integral free WordPress theme. Its strength lies in displaying content on a single page in a simple and elegant manner. It's super easy to customize and allows you to create a stunning website in minutes. It boasts a rich color scheme with light and dark sections and a clean, minimal, modern design with smooth parallax effect. Integral is built with Bootstrap, is responsive and integrates with popular plugins like Contact Form 7, Woocommerce. Integral is now compatible with Elementor & Beaver Builder page-builders. Perfect for professionals, agencies, small business, corporate business, startup, freelancer, development company, personal, portfolio, blog, real estate, lawyer, travel and photography websites.

== Installation ==

1. Upload the zip file to your themes directory.
2. Apply the theme.
3. Install Redux Framework and Contact Form 7 plugins.
4. Create page entitled "Home"
5. Set "home" page as the static front-page in your settings.
6. Go to Appearance > Customize in main menu and configure your theme's settings.
